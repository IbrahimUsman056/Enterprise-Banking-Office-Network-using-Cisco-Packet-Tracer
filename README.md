# 🏦 Enterprise Banking Office Network

## Overview

The Enterprise Banking Office Network is a Computer Networks project that focuses on designing and implementing a secure and scalable network infrastructure for a banking organization. The network connects multiple departments within the bank, enabling efficient communication, resource sharing, and secure data transmission.

The project demonstrates fundamental networking concepts such as IP addressing, subnetting, routing, switching, VLANs, network security, and client-server communication. The designed network ensures reliable connectivity between banking departments while maintaining security and performance standards required in a financial environment.

---

## Objectives

* Design a secure banking office network.
* Connect multiple departments through a structured network topology.
* Enable efficient communication and resource sharing.
* Implement logical network segmentation.
* Ensure secure and reliable data transmission.

## Features

### 🌐 Department Connectivity

Provides communication between all departments within the banking office.

### 🔒 Network Security

Implements secure communication practices to protect sensitive banking information.

### 📡 IP Addressing & Subnetting

Efficient allocation of IP addresses through subnetting techniques.

### 🔄 Routing

Facilitates communication between different network segments.

### 🖧 VLAN Segmentation

Allows administrators to monitor connectivity and network performance.
